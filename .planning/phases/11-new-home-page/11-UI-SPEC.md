## Spacing Scale

Declared values (multiples of 4 only). Source: 8-point scale standard; confirmed with existing component patterns.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding, image overlay insets |
| sm | 8px | Gap between card name and price badge, button icon gap |
| md | 16px | Default element spacing, section horizontal padding (mobile), card grid gap |
| lg | 24px | Hero inner vertical rhythm between title, subtitle, CTAs |
| xl | 32px | Hero section top/bottom padding (mobile: `py-20` = 80px handled via Tailwind scale) |
| 2xl | 48px | Section-to-section gap between hero and high-value grid |
| 3xl | 64px | Hero section top/bottom padding (desktop: `py-32` = 128px handled via Tailwind scale) |

Exceptions: NavBar height 56px (h-14 = 3.5rem) — pre-existing value, do not change. Touch target minimum 44px height for nav links (h-11 = 2.75rem) — matches existing NavBar pattern.

## Color

All tokens resolve via CSS custom properties defined in globals.css. Light and dark mode both declared.

| Role | CSS Token | Light value (oklch) | Dark value (oklch) | Usage |
|------|-----------|--------------------|--------------------|-------|
| Dominant (60%) | `--background` | `oklch(1 0 0)` (white) | `oklch(0.141 0.005 285.823)` (near-black) | Page background, hero section background |
| Secondary (30%) | `--card` / `--muted` | `oklch(1 0 0)` / `oklch(0.967 0.001 286.375)` | `oklch(0.21 0.006 285.885)` / `oklch(0.274 0.006 286.033)` | Card tiles, grid card backgrounds, loading skeleton |
| Accent (10%) | `--primary` | `oklch(0.5 0.134 242.749)` (blue) | `oklch(0.443 0.11 240.79)` (muted blue) | Primary CTA button ("Import Collection") fill; NavBar active link border-bottom; price badge text |
| Destructive | `--destructive` | `oklch(0.577 0.245 27.325)` (red) | `oklch(0.704 0.191 22.216)` (lighter red) | Not used in Phase 11 (no destructive actions on this page) |

Accent reserved for:
1. Primary CTA button fill ("Import Collection" — `variant="default"` which resolves to `--primary`)
2. NavBar active link `border-b-2 border-primary` indicator
3. Price badge foreground text on card tiles

Secondary CTAs ("Build a Deck", "Trade") use `variant="outline"` — border only, no accent fill.

Hero background treatment: Use `bg-background` (the dominant surface color). Apply a subtle top-to-bottom gradient overlay using Tailwind arbitrary values to create depth: `bg-gradient-to-b from-background via-background to-muted`. Do NOT use a background image or CDN asset for the hero — pure CSS only.

## Interaction States

### Hero CTA Buttons

| State | Visual |
|-------|--------|
| Default (primary) | bg-primary text-primary-foreground — shadcn/ui `variant="default" size="lg"` |
| Default (secondary) | border border-input bg-background — shadcn/ui `variant="outline" size="lg"` |
| Hover | shadcn/ui default hover state (opacity shift or bg-primary/90) |
| Focus | ring-2 ring-ring ring-offset-2 — shadcn/ui default focus ring |
| Active | scale-[0.98] — shadcn/ui built-in |

### CardPriceTile

| State | Visual |
|-------|--------|
| Image loading | `animate-pulse` on container div + `opacity-0` on image |
| Image loaded | Image fades in via `transition-opacity duration-300` on `onLoad` |
| Image error | `onError={() => setLoaded(true)}` — removes pulse, shows bg-muted placeholder |
| Hover | No explicit hover state required. Browser default cursor pointer via `<Link>` block. Optional: `group-hover:opacity-90 transition-opacity` on image. |
| Null price | Display `'—'` as price (confirmed by existing pattern in card detail page) |

### NavBar Brand Link

| State | Visual |
|-------|--------|
| Default | `font-heading font-semibold text-base text-foreground` |
| Hover | `hover:opacity-80 transition-opacity` |
| Active (on `/`) | No separate active style — brand is not a nav item |

## Accessibility Contract

| Element | Requirement |
|---------|-------------|
| Hero H1 | Single `<h1>` on the page; all other headings are `<h2>` or lower |
| CTA buttons | `asChild` with `<Link>` — rendered as `<a>` with visible text; no icon-only buttons |
| CardPriceTile | `<Link aria-label="View {name}">` wraps entire tile; truncated card name covered by aria-label |
| Image alt text | `alt={name}` on every `next/image` in CardPriceTile |
| Loading skeleton | `animate-pulse` is purely decorative; no aria-busy required for static server-side preloaded data |
| Color contrast | Use `text-foreground` (≥ 4.5:1 in both light and dark modes via shadcn/ui zinc tokens) |
| Focus order | Hero CTAs in DOM order (Import → Build → Trade); grid tiles in row-major order (left-right, top-bottom) |
