# Phase 15.1 Plan 01: Sort Deck Cards by Cost Ascending Summary

**One-liner:** Added cost-ascending sort (then name alpha fallback) to `mainDeck` and `sideboard` useMemo chains in deck-builder.tsx.

## What Was Built

Both `mainDeck` and `sideboard` derived arrays in `DeckBuilder` now end with a `.sort()` call that orders cards by `cost` ascending (null/undefined treated as 0), with a secondary alphabetical sort on `card.name` as a tiebreaker.

Because `groupDeckCards` receives the already-sorted `mainDeck`, each type group (Ground Units, Space Units, Upgrades, Events, Other) will naturally render cards in cost order — giving players an at-a-glance view of the deck's cost curve.

The sideboard list is also sorted the same way for consistency.
