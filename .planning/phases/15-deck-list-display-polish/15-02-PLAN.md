| Boundary | Description |
|----------|-------------|
| CDN → next/image | Art URLs from cdn.swu-db.com rendered via next/image; no user-controlled URLs |
| User tap/hover → hoveredCard state | Card rows accept touch/mouse events; state update is read-only (sets Card reference from allCards prop) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-15-02-1 | Information Disclosure | frontArtUrl in Image src | accept | CDN URLs are public; next.config.ts already has remotePatterns for cdn.swu-db.com; unoptimized:true, no signed URLs needed |
| T-15-02-2 | Spoofing | hoveredCard state manipulation | accept | State is client-only; derives from server-fetched allCards; no user input accepted for art URLs |
| T-15-02-3 | Denial of Service | Image fill rendering many cards | accept | max 50 main deck cards; only one preview at a time; next/image lazy-loading handles this |
| T-15-02-4 | Elevation of Privilege | Remove overlay dispatches SET_LEADER/SET_BASE null | accept | dispatch is a client-side reducer action; actual persistence requires explicit save action which has server-side auth |
</threat_model>
