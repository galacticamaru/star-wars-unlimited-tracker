<feature>
  <name>Deck Validation Utility</name>
  <files>src/lib/deck-validation.ts, src/lib/deck-validation.test.ts</files>
  <behavior>
    - isValid: true if Leader=1, Base=1, MainSize >= 50, No card > 3 copies (structural rule).
    - errors: Returns specific messages for missing Leader, missing Base, under 50 cards, or exceeding 3 copies.
    - warnings: Returns warning for off-aspect cards.
    - stats: Returns cost distribution (0-9+), unit type counts, and aspect counts.
  </behavior>
  <implementation>
    Create a `validateDeck` function that accepts:
    - `leader`: Card object or null
    - `base`: Card object or null
    - `mainDeck`: Array of { card: Card, quantity: number }
    - `sideboard`: Array of { card: Card, quantity: number }

    Algorithm:
    1. Count total cards in main deck.
    2. Check Leader and Base presence.
    3. Iterate through cards:
       - Aggregate quantity by `cardDefinitionId` to check 3-copy limit.
       - Check aspects against `[...leader.aspects, ...base.aspects]`.
       - Increment stats buckets (cost, unit type, arena).
    4. Return aggregated result `{ isValid, errors, warnings, stats }`.
  </implementation>
</feature>
