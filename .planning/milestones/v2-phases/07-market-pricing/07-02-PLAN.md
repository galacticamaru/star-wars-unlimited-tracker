  <files>src/lib/sync/prices.ts</files>
  <action>
    Implement `syncPrices()` function in `src/lib/sync/prices.ts`:
    - Define list of active sets: `['SOR', 'SHD', 'TWI']`.
    - Iterate through sets sequentially.
    - For each set:
      1. Fetch prices via `fetchSetPrices`.
      2. For each card in the response, perform a DB upsert into `card_definitions` mapping by `set_code` and `card_number`.
      3. Update `priceEur`, `priceUsd`, and `pricesUpdatedAt`.
      4. Wait for 2 seconds (throttle) before proceeding to the next set.
    - Return a summary of updated cards.
  </action>
  <verify>
